# data/

Runtime data generated by the application. All contents are gitignored.

| Path | Purpose |
|------|---------|
| `live.sqlite` | SQLite database for the live web UI (30 demo orders, events, slides, routing decisions) |
| `live.sqlite-shm`, `live.sqlite-wal` | SQLite WAL auxiliary files (created automatically) |
| `rag_index/` | ChromaDB vector index built from `knowledge_base/` |
| `evaluation_summaries/` | Cached evaluation summary data |

## Resetting

To reset the live database to a clean seeded state:

```bash
rm -f data/live.sqlite data/live.sqlite-shm data/live.sqlite-wal
./scripts/start_server.sh  # auto-seeds on missing DB
```

To rebuild the RAG index:

```bash
./scripts/build_rag_index.sh
```
